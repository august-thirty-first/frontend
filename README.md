### About Service
* 익명의 유저들과 함께 PingPong게임을 할 수 있음
* 닉네임과 유저 프로필 사진 커스텀 가능
* 일반게임과 레더게임을 즐길 수 있고 레더 점수가 게임 결과에 따라 증감
* 친구추가 및 친구추가 수락을 통한 친구관계 설정 가능
* 레더게임 점수 및 친구추가에 따른 다양한 숨겨진 업적 달성 가능
* 공개, 비밀 채팅방에서 원하는 유저들과 채팅을 할 수 있고 1:1DM기능 사용 가능
